# Notty - Your Everyday Notes

A modern, elegant note-taking web application with notebook organization capabilities. Built with vanilla JavaScript, offering a clean interface for managing your daily notes efficiently.

##  Features

- Notebook Organization: Create multiple notebooks to categorize your notes
- Rich Note Creation: Add, edit, and delete notes with ease
- Dark/Light Theme: Toggle between dark and light modes with persistent preferences
- Local Storage: All your data is stored locally in your browser
- Responsive Design: Works seamlessly on desktop and mobile devices
- Timestamps: Track when notes were created with relative time display
- Inline Editing: Edit notebook names directly with a simple click
- Safe Deletion: Confirmation modals prevent accidental deletions
- Material Design: Clean, modern UI following Material Design principles


### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies or build tools required!
